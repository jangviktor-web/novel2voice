# 小说转 EdgeTTS 有声配音标注脚本 — 标准化通用公式

> 本公式可无限复用，适配所有小说章节。执行顺序固定，复制即用。

---

## 一、整套流程总公式（执行顺序固定）

```
原文素材 → 角色音色映射表 → 文本分层切割规则 → 标签分段模板 → 情绪/语速参数匹配 → 输出成品配音脚本
```

---

## 二、分模块拆解标准化公式（复制即用，任何小说通用）

### 模块1：固定角色音色库公式（模板，新增角色直接追加行）

| 角色标记 | VoiceID | 基础rate | 基础pitch | 专属情绪style | 声线定位 |
|---|---|---|---|---|---|

| 【主角名】 | 对应音色ID | ±X% | ±XHz | 开心`cheerful`/发怒`angry`/温柔`gentle` | 人物定位（少年/中年/老者） |
| 【老年配角】 | `zh-CN-YunzeNeural` | -10% | -4Hz | `fearful`压抑 | 龙王、长老、老太爷 |
| 【中年稳重女性】 | `zh-CN-XiaoruiNeural` | -5% | -2Hz | `gentle` | 聪明老管家、主母 |
| 【孩童群杂】 | `zh-CN-XiaoshuangNeural`/`YunxiaNeural` | +15% | +3Hz | `chat`活泼 | 小妖、小猴、小鬼卒 |
| 【天庭官方人物】 | `zh-CN-YunyangNeural` | -7% | -3Hz | `newscast`庄重 | 玉帝、冥王、文武仙官 |
| 【温和长者】 | `zh-CN-YunjianNeural` | -5% | +1Hz | `gentle`舒缓 | 太白金星、隐士 |

| 【慌张小兵/夜叉】 | `zh-CN-YunxiaNeural` | +8% | +2Hz | `fearful` | 虾兵、鬼差、巡海夜叉 |

**规则：一个角色永久绑定一行，全书统一音色不混乱。**

### 模块2：文本切割断句公式（强制分层，AI朗读无卡顿）

#### 2.1 停顿分级符号（固定标准）

| 符号 | 场景 | 停顿时长 |
|---|---|---|
| `，` | 句内短换气 | 0.25s |
| `。！？` | 单句结束 | 0.6s |
| `：` | 人物切换/场景转场 | 1.0s |
| 空一行 | 大段落切换（龙宫→地府→天庭） | 1.5s |
| 古诗/骈文 | 每句单独换行，语速放缓 | 按句断 |

#### 2.2 文本三拆分原则（核心分割公式）

1. **叙事描写（环境、动作、风景、心理旁观）** → 全部划入【旁白】单独一段
2. **人物内心想法（XX心中暗道/暗想）** → 【角色名】单独一段，标注内心放缓
3. **人物直接对话（XX道："XXX"）** → 拆分为两段：【角色名】+ 纯台词，动作描写剥离给旁白

**示例拆分逻辑：**

原文：
```
悟空大喜，心中想到："我正好想上天看看。"
```

拆分后：
```
【旁白】
悟空大喜，/

【孙悟空｜cheerful】
心中想到：/"我正好想上天看看。"/
```

### 模块3：角色标签标准模板公式（固定格式，所有角色统一）

#### 基础格式
```
【角色标识】
纯台词/叙事文本，用 `/` 做断句分隔
```

#### 带情绪强化格式（角色有喜怒哀乐时）
```
【角色名｜情绪关键词】
文本/断句/
```

#### 完整携带参数标注（对接代码批量生成TTS专用）
```
【角色名｜VoiceID｜rate=±X%｜style=xxx】
文本/断句/
```

### 模块4：对话分离标准公式（处理"XX道："句式万能拆解）

**输入原文句式：** `A动作，A道："台词内容"`

**拆解固定输出：**
1. 动作部分 → 【旁白】单独一段
2. 说话人标识 → 【A】单独标签段
3. 引号内台词 → 跟在【A】标签下，逐句`/`断句

**示例拆解演算：**

原文：
```
龙王吓得浑身发抖，慌忙道："上仙切莫动手！"
```

套用公式输出：
```
【旁白】
龙王吓得浑身发抖，/

【东海龙王敖广｜fearful】
慌忙道：/"上仙切莫动手！"/
```

---

### 模块5：群杂人物处理公式（多人喧闹、小妖小猴）

1. 多名孩童/小妖统一使用【群杂】标签，交替分配男女童音色
2. 多人齐声说话：合并一段，语速+15%，标注`chat`活泼情绪
3. 多人先后发言：每换一个人，新建一段【群杂】分段

---

### 模块6：诗词、奏折、祭文专用处理公式

1. 全部归属【旁白】，单独起段，标注`poetry-reading`风格
2. 每一句诗词单独换行，末尾加`/`
3. 长篇公文（龙王上表、地府文书）整段旁白平缓朗读，不加角色音

## 三、一键套用完整输出示例（公式落地成品）

**输入原文短句：**
```
四老猴上前说道："大王，水下直通东海龙宫，可去求取兵器。"
```

**执行步骤：**
1. 查表匹配角色：【四老猴】`zh-CN-XiaoruiNeural` rate=-5% gentle
2. 拆分动作+对话：动作归旁白，台词归角色
3. 断句加分隔符`/`
4. 套标签模板输出：

```
【旁白】
四老猴上前，/

【四老猴｜zh-CN-XiaoruiNeural｜rate=-5%｜style=gentle】
说道：/"大王，/ 水下直通东海龙宫，/ 可去求取兵器。"/
```

## 四、批量制作自动化执行步骤（重复落地流程）

1. 新建文档顶部粘贴【角色音色配置表】，全书统一音色
2. 复制小说原文，逐段扫描区分三类内容：旁白动作、内心独白、人物对话
3. 按对话拆分公式切割动作与台词，分开分段
4. 给每一段打上固定【角色】标签，匹配表格内rate、style参数
5. 全文字句统一用`/`进行精细化断句，诗词单行拆分
6. 大场景切换处空一行做长停顿区分
7. 导出文本，一段一个角色，分批次输入EdgeTTS生成音频，后期拼接

---

## 五、避错约束公式（强制规则，防止配音混乱）

1. **一段文本只能有一个角色标签**，禁止同一段混合多个人物台词
2. 人物动作、神态、环境描写**绝不允许**混入角色对话段落
3. 同一个角色**全书固定一套音色、语速**，不随意更换
4. 争吵、发怒人物统一添加 `style=angry`；胆小害怕统一 `style=fearful`；开心活泼统一 `cheerful`
5. 内心独白**必须单独一段**，不和外部对话合并

## 六、与 generate_audiobook.py 的对接方式

本标注公式产出的文本可直接用于 `--auto` 模式：

```bash
# 使用标注后的文本生成有声书
python3 {baseDir}/scripts/generate_audiobook.py \
  --auto annotated_novel.txt \
  output_dir \
  --tts-backend edge

# 或手动指定音色覆盖
python3 {baseDir}/scripts/generate_audiobook.py \
  --auto novel.txt \
  output_dir \
  --voices '{"旁白":"zh-CN-XiaoxiaoNeural","孙悟空":"zh-CN-YunxiNeural"}' \
  --tts-backend edge
```

脚本会自动识别 `【角色名】` 标签格式，并按标注的 rate/pitch/style 参数调用 Edge TTS 生成。
