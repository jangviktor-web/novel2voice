Quick Start

三步开始，无需 API Key

# 1. 安装
openclaw skills install @jangviktor-web/novel2voice

# 2. 生成有声书（自动识别角色）
python3 scripts/generate_audiobook.py --auto novel.txt ./output

# 3. 异步并发（推荐，3-5x 加速）
python3 scripts/generate_audiobook.py --auto novel.txt ./output --async --concurrent 5

FAQ

常见问题

需要 API Key 吗？
默认使用 Edge TTS，完全免费，无需任何 Key。如果需要更高质量的音色（如音色复刻、唱歌），可以配置 MiMo TTS 的 API Key。
支持哪些操作系统？
Linux、macOS、Windows 全平台支持。只需要 Python 3.8+ 和 FFmpeg。
生成速度如何？
使用 --async --concurrent 5 参数，5 线程并发生成。一本 10 万字的小说大约需要 10-20 分钟（取决于网络速度）。
角色识别不准怎么办？
可以通过 --character-voices 参数手动指定角色→音色映射，例如：--character-voices '{"旁白":"云扬","张三":"云希"}'。
支持哪些字幕格式？
支持 SRT 和 ASS/SSA 格式，自动检测文件编码（UTF-8、GBK、Big5、Shift_JIS 等），自动修复重叠时间码。
可以作为 Agent Skill 使用吗？
可以。Novel2Voice 遵循 Agent Skills 标准，兼容 QoderWork、OpenClaw、Claude Code 等 skills-compatible runtime。一行命令安装：openclaw skills install @jangviktor-web/novel2voice
