Features

核心功能

LLM 驱动的智能角色识别，让每本小说都有声音

🎭

多角色自动识别

LLM 分析角色性别、年龄、性格，自动匹配最佳音色

🎤

500+ 高质量音色

Edge TTS 覆盖 100+ 语言/地区，MiMo TTS 提供小米音色

🎨

50+ 情感风格控制

温柔/愤怒/悲伤/撒娇等行内标签，支持方言、唱歌模式

⚡

异步并发生成

多线程 TTS，速度提升 3-5x，支持断点续传

📝

字幕完美对齐

自动检测 SRT/ASS 编码，按时间轴精确生成

🎵

章节智能拆分

自动识别「第X章/回/节」，按章合并输出

Quick Start

快速开始

3 步完成安装和使用

1

ClawHub 一键安装（推荐）

skills-compatible runtime

openclaw skills install @jangviktor-web/novel2voice
2

直接克隆

git clone https://github.com/jangviktor-web/novel2voice.git cd novel2voice pip install requests charset_normalizer
3

一键安装脚本

curl -fsSL https://raw.githubusercontent.com/jangviktor-web/novel2voice/main/install.sh | bash

依赖检查

python3 --version ≥ 3.8   ·   ffmpeg -version 已安装
