核心功能

🎭

多角色自动识别

LLM 分析角色性别、年龄、性格，自动匹配最佳音色

🎤

500+ 高质量音色

Edge TTS 覆盖 100+ 语言/地区，MiMo TTS 提供小米音色

🎨

50+ 情感风格控制

温柔/愤怒/悲伤/撒娇等行内标签，支持方言、唱歌模式

⚡

异步并发生成

多线程 TTS，速度提升 3-5x，支持断点续传

📝

字幕完美对齐

自动检测 SRT/ASS 编码，按时间轴精确生成

🎵

章节智能拆分

自动识别「第X章/回/节」，按章合并输出

快速开始

1

安装依赖

安装 FFmpeg 和 Python 依赖

# 安装 FFmpeg
apt install ffmpeg  # Ubuntu/Debian
brew install ffmpeg # macOS

# 安装 Python 依赖
pip install requests charset_normalizer
2

生成有声书

自动模式：LLM 识别角色 → 分配音色 → 生成

python3 scripts/generate_audiobook.py --auto novel.txt ./output
3

查看结果

生成文件位于 output 目录

output/
├── output_complete.mp3
├── chapters/
│   ├── 001_第一章.mp3
│   └── 002_第二章.mp3
